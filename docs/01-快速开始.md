# 快速开始

本指南帮助你在 5 分钟内开始使用 Craft Companion。

## 第一步：先分清你是哪种用户

### 方式一：从零开始
适合：还没有正文，准备和 AI 一起搭建新小说。

### 方式二：导入已有小说
适合：已经有章节文本，想迁移到 Craft Companion。

---

## 第二步：创建项目

```bash
node tools/init.js
```

初始化时系统会问你：
1. 项目名称
2. 你是“从零开始”还是“导入已有小说”

创建完成后，进入项目目录：

```bash
cd 你的项目名
```

然后先看：

```text
START_HERE.md
```

---

## 如果你是“从零开始”

按顺序使用这些模板：

- `提示模板/从零开始/01-定义核心概念.md`
- `提示模板/从零开始/02-构建主角.md`
- `提示模板/从零开始/03-设计故事结构.md`
- `提示模板/从零开始/04-建立文风.md`
- `提示模板/从零开始/05-准备开始创作.md`

完成后再进入正式创作。

---

## 如果你是“导入已有小说”

按顺序使用这些模板：

- `提示模板/导入已有小说/01-提取人物信息.md`
- `提示模板/导入已有小说/02-提取世界观设定.md`
- `提示模板/导入已有小说/03-构建时间线.md`
- `提示模板/导入已有小说/04-分析文风特征.md`
- `提示模板/导入已有小说/05-识别伏笔线索.md`

完成后再进入正式创作。

---

## 5 阶段工作流

Craft Companion 当前统一使用 **5 阶段工作流**：

1. 章纲
2. 初稿
3. 自查
4. 修订
5. 终版确认

如果你在其他文件里看到旧的 6 阶段说法，以这里为准，后续会持续统一。

---

## 常用命令

```bash
# 创建新项目
node tools/init.js

# 显示导入已有小说的模板
node ../tools/import-cli.js

# 查看命令帮助
craft-companion --help
```

---

## 最重要的三件事

1. 先分清自己走哪条路：从零开始 / 导入已有小说
2. 先完成模板初始化，再写正文
3. 统一使用 `提示模板/...` 路径，不再使用旧 `prompts/...`
